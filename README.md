# 🎓 APSSDC Internship Attendance Analysis

## 📌 Project Overview

This project analyzes attendance records of students participating in an APSSDC (Andhra Pradesh State Skill Development Corporation) internship program. Using Python and data analysis techniques, the project processes attendance data, calculates attendance percentages, identifies eligible students based on attendance requirements, and visualizes attendance patterns.

The main objective is to help understand attendance performance and determine which students satisfy the minimum attendance criteria for internship participation.

---

## 🚀 Key Features

* Import attendance data from multiple CSV files
* Merge and clean internship attendance records
* Remove duplicate entries
* Perform exploratory data analysis (EDA)
* Calculate attendance percentages
* Identify eligible and non-eligible students
* Generate attendance visualizations
* Analyze attendance trends across participants

---

## 📂 Project Workflow

### Step 1: Data Collection

Attendance records from APSSDC internship sessions are collected and loaded into Python using Pandas.

### Step 2: Data Cleaning

* Inspect dataset structure
* Check for missing values
* Remove duplicate records
* Drop unnecessary columns

### Step 3: Data Analysis

* Generate summary statistics
* Analyze attendance-related metrics
* Explore relationships between attendance variables

### Step 4: Attendance Percentage Calculation

Attendance percentage is calculated for each participant based on session participation duration.

### Step 5: Eligibility Check

Students are categorized as:

* **Eligible** → Attendance ≥ 80%
* **Not Eligible** → Attendance < 80%

### Step 6: Visualization

The project generates:

* Attendance distribution charts
* Eligibility comparison graphs
* Correlation heatmaps
